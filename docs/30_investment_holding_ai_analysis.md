# 銘柄別AI分析

## 目的

投資タブの銘柄カードから、1銘柄だけを対象にしたAI分析プロンプトをコピーし、AI分析履歴へ保存できるようにする。AI APIには接続せず、ユーザーがChatGPTなどへ貼り付けて学習・検討に使う。

## 全体投資分析との違い

全体投資分析は、総評価額、現金比率、資産配分、実保有・仮想保有の全体像を確認する。

銘柄別分析は、1銘柄の評価額、評価損益、ポートフォリオ内の比率、PER / PBR / ROE / 配当利回り、追加で調べるべき情報を深掘りする。

## 銘柄カードからの流れ

1. 投資タブで銘柄カードを表示する
2. 「銘柄AI分析」ボタンを押す
3. 対象銘柄の分析payloadを作成する
4. 銘柄別AI分析プロンプトを作成する
5. クリップボードへコピーする
6. AI分析履歴へ保存する

## AI分析履歴への保存

`AiAnalysisRun`として保存する。

- `theme`: `investment_holding_review`
- `source`: `investment_holding_card`
- `status`: `prompt_copied`
- `title`: `${銘柄名} 銘柄AI分析`
- `promptText`: コピーしたプロンプト全文
- `payloadJson`: 銘柄別分析payload

## PER / PBR / ROE / 配当利回りとの関係

銘柄別payloadには、投資指標学習カードと同じ`buildInvestmentIndicatorReport`から対象銘柄の指標insightsを含める。指標判定ロジックを重複させず、画面表示、投資JSON、AIプロンプトで同じ確認観点を使う。

## 現金データの扱い

`assetType === "cash"` のデータではPER / PBR / ROE / 配当利回りの銘柄分析を行わない。現金比率、生活防衛資金、投資待機資金、追加投資を急がないための確認観点として扱う。

## 注意

AI分析は学習・検討用であり、投資判断を断定するものではない。プロンプトでも「買うべき」「売るべき」と断定しないよう明記する。株価・指標は手入力またはサンプルであり、株価API、財務データAPI、証券口座API、AI APIには接続しない。

## 今回できること

- 銘柄カードから銘柄別AI分析プロンプトをコピーできる
- コピー時にAI分析履歴へ保存できる
- 学習・AI分析画面で「銘柄分析」ラベルを表示できる
- 銘柄分析結果、メモ、次の行動を既存のAI分析履歴UIで保存できる
- 銘柄分析履歴を削除できる
