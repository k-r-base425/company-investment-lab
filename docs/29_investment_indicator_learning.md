# 投資指標学習

## 目的

投資タブで保存済み銘柄のPER / PBR / ROE / 配当利回りを確認し、銘柄データと投資指標の意味を結びつけて学べるようにする。

## 対象指標

- PER: 株価が利益の何倍まで買われているかを見る指標
- PBR: 株価が純資産の何倍まで買われているかを見る指標
- ROE: 自己資本を使ってどれだけ利益を出しているかを見る指標
- 配当利回り: 投資額に対して年間配当がどれくらいあるかを見る指標

## 簡易判定ルール

現金は銘柄分析対象外とし、cash以外の保存済み投資データを対象にする。

- PER未入力はunknown、10未満はnotice、10以上25以下はgood、25超はwarning
- PBR未入力はunknown、1未満はnotice、1以上3以下はgood、3超はwarning
- ROE未入力はunknown、5%未満はwarning、5%以上10%未満はnotice、10%以上はgood
- 配当利回り未入力はunknown、0%または2%未満はnotice、2%以上5%以下はgood、5%超はwarning

## 断定しない方針

指標は学習・分析用の確認材料として扱う。画面やAI分析プロンプトでは「買うべき」「売るべき」「割安です」のような断定を避ける。

代わりに、以下のような表現を使う。

- 確認候補です
- 注意して見たい水準です
- 業種比較が必要です
- 他の指標と合わせて確認しましょう

## 業種比較や過去推移が必要な理由

PER、PBR、ROE、配当利回りは業種、成長性、財務状態、相場環境によって目安が変わる。単年の数値だけで判断せず、業種平均、過去推移、利益成長、負債水準、配当方針などと合わせて確認する。

## 実保有と仮想保有

実保有は現在の保有状況を確認するために使う。仮想保有は学習用ポートフォリオとして、実際に投資する前に指標や値動きを確認するために使う。

## AI分析JSON / 投資JSONエクスポート

投資分析payloadに`investmentIndicatorReport`を含める。

- `topics`: 指標の意味、計算式、読み方、注意点
- `insights`: 銘柄ごとの指標判定、表示値、確認ポイント
- `holdingCount`: cash以外の判定対象銘柄数

投資JSONエクスポートと投資AI分析プロンプトのJSONは同じ判定ロジックを使う。

## 今回できること

- warning / unknownを優先して要確認ポイントを表示できる
- 投資JSONエクスポートと投資AI分析プロンプトに指標レポートを含められる
