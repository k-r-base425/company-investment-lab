# 会計タブ表示整理

## 目的

会計タブは、入力、前月比、月比較、カテゴリ分析、改善コメント、改善アクション、改善効果、出力導線が増えたことで縦に長くなっていました。

今回の整理では、新しい分析機能は追加せず、既存機能を「概況」「分析」「入力」「出力」の4つの折りたたみセクションにまとめ、必要な場所へ移動しやすくします。

## 4つのグループ構成

### 概況

選択月の主要な数字を確認するセクションです。

- 今月の入力サマリー
- 前月比カード
- 月比較セクション

初期状態は開いています。

### 分析

カテゴリ、改善コメント、改善アクション、改善進捗を確認するセクションです。

- カテゴリ別前月比
- 改善コメント
- 改善アクション
- 改善効果トラッキング
- カテゴリ分析

初期状態は開いています。

### 入力

売上、経費、家計、仕訳を追加・編集するセクションです。

- 入力タイプ切り替え
- 入力フォーム
- 最近の入力リスト

初期状態は開いています。

### 出力

CSV、JSON、AI分析プロンプトの出力導線を確認するセクションです。

- データ出力カード

初期状態は閉じています。

## 折りたたみUI

`CollapsibleSection` は会計画面内の状態で開閉を管理します。

- ヘッダーを押すと開閉する
- 開いているときだけ内容を表示する
- タイトル、サブタイトル、バッジ、開閉アイコンを表示する
- 開閉状態は永続化しない

## クイックナビ

会計タブ上部に「概況」「分析」「入力」「出力」のチップを表示します。

- 押したセクションを開く
- 他のセクションは閉じない
- スクロール制御は行わない
- スマホ幅で折り返して表示する

## 今回できること

- 会計タブの主要機能を4グループに整理できる
- 各セクションを折りたたんで表示量を減らせる
- クイックナビから主要セクションを開ける
- 既存の入力、分析、改善アクション、出力導線を維持できる

## 今回やらないこと

- 新しい分析機能の追加
- 下部タブの追加
- 大規模なルーティング変更
- セクション開閉状態の永続化
- API連携
- SQLiteやlocalStorageの保存スキーマ変更
