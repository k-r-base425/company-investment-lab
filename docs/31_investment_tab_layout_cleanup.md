# 投資タブ表示整理

## 目的

投資タブに投資サマリー、保有銘柄、投資指標学習、銘柄別AI分析、投資データ出力が増えたため、画面を4つの折りたたみセクションに整理して見通しを改善する。

## 背景

投資タブでは、ローカル保存、追加・編集・削除、ホームKPI連動、投資データ出力、投資AI分析履歴、投資指標学習、銘柄別AI分析の導線を同じ画面に置いている。スマホ幅では縦に長くなるため、会計タブと同じ折りたたみ方式で整理する。

## 4つのグループ構成

1. 概況
2. 保有
3. 学習・分析
4. 入力・出力

## 概況セクション

投資全体の評価額・損益・資産配分を確認する。

- 投資データ状態表示
- 投資サマリー
- 資産配分カード

初期状態は開いている。

## 保有セクション

実保有・仮想保有の銘柄を確認する。

- 実保有 / 仮想保有 / すべて切り替え
- 保有銘柄リスト
- 編集
- 削除
- 銘柄AI分析

初期状態は開いている。

## 学習・分析セクション

投資指標とAI分析の確認ポイントを学ぶ。

- 銘柄別AI分析の補足説明
- 投資指標学習セクション
- 投資指標メモカード
- 投資判断を断定しない注意書き

初期状態は閉じている。

## 入力・出力セクション

銘柄追加と投資データ出力を行う。

- 銘柄追加 / 編集フォーム
- 投資データ出力カード

初期状態は開いている。

## 折りたたみUI

会計タブと同じ`CollapsibleSection`を再利用する。

- 白背景カード
- 境界線
- 角丸
- ヘッダー押下で開閉
- バッジ表示
- 開閉アイコン表示
- スマホ幅で横にはみ出さない

## クイックナビ

投資タブ上部に`InvestmentQuickNav`を追加する。

- 概況
- 保有
- 学習
- 入出力

押したセクションが閉じている場合は開く。他セクションは閉じない。スクロール制御や永続化は今回行わない。

## 今回できること

- 投資タブを4セクションに整理できる
- 各セクションを開閉できる
- クイックナビから主要セクションを開ける
- 既存の投資保存・編集・削除、出力、投資指標学習、銘柄別AI分析を維持できる

## 今回やらないこと

- 下部タブ追加
- 銘柄詳細画面

- セクション開閉状態の保存
