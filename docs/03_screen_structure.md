# 画面構成

## 5つの下部タブ

1. ホーム
2. 会計
3. 投資
4. 学習
5. 設定

## 各画面の目的

### ホーム

毎日最初に見るダッシュボード。会計、家計、投資、学習、AI分析への入口を集約する。

### 会計

事業、副業、疑似株式会社モードの取引を入力し、売上、経費、利益、税金目安を確認する。

### 投資

実保有資産と仮想保有銘柄を管理し、評価額、損益、資産配分、投資可能額を確認する。

### 学習

簿記2級、会計、投資指標、家計、経営判断の学習テーマを管理し、日々の学習を継続する。

### 設定

カテゴリ、税金目安、表示、出力、将来のAPIキー項目など、アプリ全体の設定を管理する。

## 各画面の主要機能

### ホーム

- 今月の売上、経費、利益、税金目安
- 投資可能額、総資産、現金比率、投資損益
- 学習進捗
- 売上・経費・利益の推移
- 資産配分
- 今日の学習
- AI分析用データ生成カード
- 会計入力ショートカット
- 家計簿ショートカット
- 投資分析ショートカット
- CSV出力ショートカット

### 会計

- 会計取引の追加、編集、削除
- 売上、経費、入金、出金の分類
- 勘定科目、日付、金額、メモ、タグの入力
- 月次の売上、経費、利益、税金目安の表示
- 疑似株式会社モード用の経営判断メモ

### 投資

- 実保有資産の登録
- 仮想保有銘柄の登録
- 数量、取得価格、現在価格、評価額、損益の管理
- 資産配分の確認
- 投資分析用データ生成

### 学習

- 学習テーマ一覧
- 今日の学習
- 進捗率、理解度、復習予定
- 学習メモ
- 会計、投資、家計、経営判断のカテゴリ管理

### 設定

- 基本表示設定
- 税金目安の概算設定
- カテゴリ設定
- CSV/JSON出力設定
- 将来API連携用の空項目
- データバックアップ方針の確認

## ホーム画面の詳細構成

ホーム画面は、情報量の多いスマホダッシュボードを基本にする。小さなカードを複数配置し、重要な数字を一画面で確認できるようにする。

### 上部

- アプリ名または画面名
- 今月のサマリー
- 通知または設定への小さな導線

### 主要指標カード

- 売上
- 経費
- 利益
- 税金目安
- 投資可能額
- 総資産
- 現金比率
- 投資損益

### グラフ領域

- 売上・経費・利益の推移
- 資産配分

### 学習領域

- 学習進捗
- 今日の学習テーマ
- 次に復習するテーマ

### ショートカット領域

- 会計入力
- 家計簿入力
- 投資分析
- CSV出力
- AI分析用データ生成

## AI分析カードの位置づけ

AI分析カードは、ホーム画面内で最も目立つ行動導線のひとつにする。

役割は、アプリ内の実データと疑似データをAIに渡せる形へ変換すること。MVPではAI APIを使わず、生成されたJSONとプロンプトをユーザーがコピーしてChatGPTなどに貼り付ける。

AI分析カードには以下を含める。

- 目立つグラデーションボタン
- 分析できる内容の短い説明
- 対象データの概要
- 「AI分析用データを生成」アクション
- 生成後のコピー導線

AI分析カードは、単なる装飾ではなく、会計、家計、投資、学習を横断して判断材料に変える中心機能として扱う。
