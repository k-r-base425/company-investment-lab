# 学習タブ統合

## 目的

学習タブを「学習・AI分析」画面として整理し、会計学習、投資学習、AI分析履歴を同じ場所で確認できるようにする。

## 統合する理由

会計タブと投資タブには、入力・分析・AIプロンプト出力・AI分析履歴保存が増えている。学習タブでは、それらを横断して「どの数字をどう読むか」「AI分析結果をどう見返すか」を整理する。

## 学習ダッシュボード

学習ダッシュボードでは、対象月の学習状況とAI分析履歴の件数を確認する。

- 会計学習テーマ数
- 投資学習テーマ数
- AI分析履歴件数
- 回答保存済み件数
- 回答未保存件数
- 対象月

今日のおすすめ学習として、会計、投資、AI分析履歴から3件を表示する。

## 会計学習セクション

簿記2級、個人事業、経営判断に必要な数字を学ぶ。

- 利益とキャッシュフローの違い
- 簿記2級：固定資産と減価償却
- 経費率と利益率
- 仕訳の基本
- 税金目安と投資可能額

## 投資学習セクション

実保有、仮想保有、投資指標を使って銘柄を見る力を鍛える。

- PER / PBR / ROE
- 配当利回り
- 現金比率と安全余力
- 実保有と仮想保有
- 銘柄別AI分析

## AI分析履歴セクション

既存のAI分析履歴機能をそのまま利用する。

- AI分析履歴一覧
- 回答保存フォーム
- メモ
- 次の行動
- プロンプト再コピー
- 削除
- フィルター

## AI分析履歴の分類

履歴カードでは以下のラベルを表示する。

- `monthly_review` / `business_profitability` / `household_review`: 会計分析
- `investment_review`: 投資分析
- `investment_holding_review`: 銘柄分析
- `learning_review`: 学習
- その他: その他

フィルターは、すべて、会計、投資、銘柄、回答保存済み、回答未保存を表示する。

## 学習カード仕様

学習カードは以下を表示する。

- カテゴリバッジ
- タイトル
- 説明
- 関連画面
- 学習ポイント

長い日本語は折り返し、スマホ幅で横にはみ出さないようにする。
